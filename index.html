🚀 Começar Rápido

  1. Banco de Dados: Copie SQL_COMPLETO.sql e execute no PostgreSQL
  2. Instalar: npm install
  3. Rodar: npm run dev
  4. Acessar: http://localhost:3000

🔄 Fluxo de Atendimento

  1. Paciente chega na clínica
  2. Emite senha no totem ou escaneia QR Code
  3. Acompanha pelo celular ou painel da TV
  4. Recepção chama o paciente
  5. Paciente é atendido na recepção
