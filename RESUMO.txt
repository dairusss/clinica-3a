========================================
🚀 COMO COMEÇAR
========================================

1️⃣  SETUP POSTGRESQL

   Opção A - Interface (pgAdmin):
   - Abra pgAdmin
   - Crie novo banco chamado "clinica"
   - Copie e cole todo o conteúdo de SQL_COMPLETO.sql
   - Execute

   Opção B - Terminal (psql):
   $ psql -U postgres
   postgres=# CREATE DATABASE clinica;
   postgres=# \c clinica
   clinica=# \i SQL_COMPLETO.sql

2️⃣  INSTALAR DEPENDÊNCIAS

   $ cd C:\Users\Weyffer\Desktop\Clinica
   $ npm install

3️⃣  CONFIGURAR ARQUIVO .env

   $ copy .env.example .env
   
   Editar .env:
   DB_HOST=localhost
   DB_PORT=5432
   DB_USER=postgres       (ou seu usuário)
   DB_PASSWORD=sua_senha
   DB_NAME=clinica
   PORT=3000

4️⃣  INICIAR SERVIDOR

   $ npm run dev

5️⃣  TESTAR API

   No navegador ou Postman:
   http://localhost:3000
   
   Exemplos:
   - POST http://localhost:3000/api/pacientes/emitir-senha
   - GET  http://localhost:3000/api/pacientes/fila
   - POST http://localhost:3000/api/recepcao/chamar-proximo

📡 ENDPOINTS DISPONÍVEIS
========================================

PACIENTES:
  📝 POST   /api/pacientes/emitir-senha              → Gerar nova senha
  👁️  GET    /api/pacientes/status/:numeroSenha      → Ver status

RECEPÇÃO:

  ✅ POST   /api/recepcao/finalizar/:numeroSenha    → Finalizar atendimento
  📋 GET    /api/recepcao/historico                 → Histórico de chamadas
  📊 GET    /api/recepcao/estatisticas              → Stats do dia

========================================
📚 FLUXO DO SISTEMA
========================================

1. Paciente chega na clínica
   └─> Toca no totem ou escaneia QR Code

2. Emitir senha
   └─> POST /api/pacientes/emitir-senha
   └─> Retorna: número "0047"

3. Acompanhar na fila
   └─> GET /api/pacientes/status/0047
   └─> Mostra: posição na fila + tempo de espera

4. Painel de TV + Celular
   └─> GET /api/pacientes/fila (atualiza a cada 5s)
   └─> Mostra todos na fila

5. Recepção chama paciente
   └─> POST /api/recepcao/chamar-proximo
   └─> OU POST /api/recepcao/chamar/0047
   └─> Painel exibe: "SENHA 0047 - RECEPÇÃO"

6. Recepcionista inicia atendimento
   └─> POST /api/recepcao/iniciar/0047
   └─> Status muda para: EM_ATENDIMENTO

7. Finalizar atendimento
   └─> POST /api/recepcao/finalizar/0047
   └─> Status muda para: ATENDIDO
   └─> Próximo paciente é chamado

🧪 TESTANDO A API
========================================

Com cURL:
  curl -X POST http://localhost:3000/api/pacientes/emitir-senha \
    -H "Content-Type: application/json" \
    -d '{"nome":"João Silva","cpf":"12345678901"}'

Com Postman:
  1. Criar nova requisição POST
  2. URL: http://localhost:3000/api/pacientes/emitir-senha
  3. Body (raw JSON):
     {
       "nome": "João Silva",
       "cpf": "12345678901",
       "telefone": "11987654321"
     }
  4. Clique Send

Com JavaScript (navegador):
  fetch('http://localhost:3000/api/pacientes/emitir-senha', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ nome: 'João Silva', cpf: '12345678901' })
  })
  .then(r => r.json())
  .then(data => console.log(data));

========================================
🐛 TROUBLESHOOTING
========================================

❌ "Cannot connect to database"
   → PostgreSQL não está rodando
   → Inicie o PostgreSQL

❌ "Database clinica does not exist"
   → Execute SQL_COMPLETO.sql no PostgreSQL
   → Ou execute: psql -U postgres -d clinica -f SQL_COMPLETO.sql

❌ "Cannot find module 'express'"
   → npm install

❌ "Port 3000 already in use"
   → Altere PORT no arquivo .env

❌ "Invalid CPF"
   → CPF deve ter 11 dígitos sem pontos: "12345678901"
