🚀 COMO COMEÇAR
================================================================================

1. INICIAR SERVIDOR
   $ npm run dev

2. ABRIR TOTEM (Emissão de Senhas)
   http://localhost:3000/totem

3. PREENCHER DADOS E EMITIR SENHA
   - Recebe número + QR Code

4. ACOMPANHAR NO CELULAR
   http://localhost:3000/cliente-mobile
   - Escaneia QR Code OU digita número
   - Vê posição na fila

5. CHAMAR PACIENTE NA RECEPÇÃO
   http://localhost:3000/docs
   - POST /api/recepcao/chamar-proximo

6. MONITOR TV MOSTRA FILA
   http://localhost:3000/painel
   - Atualiza automático
